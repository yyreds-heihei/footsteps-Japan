Footsteps 散歩動画アプリ

日本各地のお散歩動画をプレイリスト別に管理し、視聴進捗を追跡できます。

📋 YouTube API キーの取得方法：
1. Google Cloud Console を開く
2. プロジェクトを作成（または選択）
3. 「APIとサービス」→「ライブラリ」→「YouTube Data API v3」を有効化
4. 「認証情報」→「APIキーを作成」でキーをコピー
▶ チャンネルIDを直接入力する（@ハンドルで見つからない場合）
YouTubeチャンネルページ →「詳細情報」→「チャンネルIDをコピー」で取得できます（例: UCxxxxxxxxxx）
